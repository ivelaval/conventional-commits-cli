## Installation

To use Git Cook, all you need to do is install the
`@vennet/gitcook` package:

```sh
# with yarn
$ yarn add @vennet/gitcook -g

# with npm
$ npm i @vennet/gitcook -g

# with pnpm
$ pnpm add @vennet/gitcook -g

# with bun
$ bun add @vennet/gitcook -g
```
